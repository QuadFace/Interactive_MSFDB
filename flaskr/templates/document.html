Operating System

The hosts are seperated into three column based on the Operating System.

  • Windows
  • Linux
  • Cisco

Purpose Icon

Each icon represent a host based on different purposes

  • Authentication
  • Firewall
  • Database
  • Proxy
  • Router

Filter

Use a keyword, '=', then a value or range of values to filter hosts.

  • ip: ip address. This can be entered with regex or CIDR notation
  • ports: a port or range of ports that are open
  • num_ports: number of open ports
  • OS: operating system
  • Examples
    • OS=Windows
    • ports=80
    • num_ports=3

File

Load in or save a database file.

View

Organize the hosts in descending order of available ports open or by host purpose.
